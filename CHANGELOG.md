### ✨ 新增功能

- **深色模式** - 自动跟随系统深色/浅色模式切换，Apple 风格配色

### 🔧 优化

- 全站颜色改为 CSS 变量，统一管理
- 导航栏、页脚、各页面适配深色模式
- 生产环境禁用管理后台路由

## [1.2.0] - 2025-12-06

### ✨ 新增功能

- **图片懒加载组件** - 新增 LazyImage 组件，支持骨架屏加载动画
- **图片尺寸预留** - 根据图片宽高比预留空间，避免布局跳动
- **不蒜子访问统计** - 页脚显示今日访问和总访问次数

### 🔧 优化

- **作品集瀑布流** - 作品集页面改为瀑布流布局，适配不同尺寸图片
- **上传自动记录尺寸** - 上传图片时自动保存宽高信息

### 🛠️ 新增脚本

- `scripts/updateImageDimensions.js` - 为现有图片补充尺寸信息

## [1.0.0] - 2025-12-06

### 🎉 首次发布

这是 Plog 的第一个正式版本，包含完整的个人摄影作品展示功能。

### ✨ 新增功能

#### 前端展示
- **首页** - 瀑布流照片墙，支持紧凑/广阔视图切换
- **作品集** - 分类筛选，统计卡片，分类描述横幅
- **关于我** - 个人简介，摄影器材，拍摄足迹地图
- **联系方式** - 微信二维码，邮箱联系
- **照片弹窗** - 高清原图查看，EXIF 信息展示，全屏模式

#### 管理后台
- **图片管理** - 上传、编辑、删除照片
- **分类管理** - 添加、编辑、删除分类
- **自动缩略图** - 上传时自动生成 800px 缩略图
- **EXIF 提取** - 自动读取相机信息和 GPS 位置
- **文件同步删除** - 删除记录时同步删除图片文件

#### 配置系统
- **统一配置文件** - `src/config/index.js` 集中管理所有配置
- **配置分类** - 站点配置、API 配置、路径配置、页面配置等
- **构建时注入** - index.html 自动读取配置（vite-plugin-html）
- **SEO 文件生成** - robots.txt 和 sitemap.xml 自动生成

#### SEO 优化
- **动态 Meta** - 每个页面独立的标题、描述、关键词
- **Open Graph** - 微信分享优化
- **结构化数据** - JSON-LD 支持
- **Canonical URL** - 规范链接

#### 性能优化
- **图片懒加载** - 延迟加载提升首屏速度
- **代码分割** - 管理后台独立打包
- **缩略图分离** - 列表用小图，详情用原图

### 🔧 技术实现

- Vue 3 + Composition API
- Vite 5 构建工具
- Tailwind CSS 原子化样式
- Element Plus 管理后台 UI
- Express 后端服务
- Sharp 图片处理
- exifr EXIF 读取
- 高德地图 API GPS 逆地理编码

### 📁 项目结构

```
plog/
├── src/
│   ├── components/      # 公共组件
│   ├── views/           # 页面视图
│   ├── config/          # 配置文件
│   ├── data/            # 数据文件
│   ├── composables/     # 组合式函数
│   └── router/          # 路由配置
├── admin-server/        # 管理后台服务
├── uploads/             # 上传的图片
├── scripts/             # 工具脚本
└── public/              # 静态资源
```

### 📝 配置项

| 配置 | 说明 |
|-----|------|
| `siteConfig` | 站点基本信息、SEO、域名 |
| `apiConfig` | 第三方 API 密钥 |
| `pathConfig` | 文件和目录路径 |
| `imageConfig` | 图片处理参数 |
| `exifConfig` | EXIF 读取配置 |
| `pageConfig` | 各页面显示内容 |
| `socialConfig` | 社交链接 |
| `navConfig` | 导航菜单 |
| `footerConfig` | 页脚配置 |

### 🛠️ 脚本命令

| 命令 | 说明 |
|-----|------|
| `npm run dev` | 启动开发服务 |
| `npm run build` | 构建生产版本 |
| `npm run build:seo` | 生成 SEO 文件 |
| `npm run preview` | 预览构建结果 |
