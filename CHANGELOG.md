## [1.9.0] - 2024-12-20

### 🎨 后台管理系统全面改版 - Apple 液态玻璃风格

#### 整体设计
- **配色系统**
  - 浅色模式：iOS 蓝 (#007aff) + Apple 灰背景 (#f5f5f7)
  - 深色模式：纯黑背景 (#000000) + 深灰卡片 (#1c1c1e)
  - 统一的 CSS 变量系统，支持主题切换

- **主题切换功能**
  - 顶部导航栏主题切换按钮（太阳/月亮图标）
  - 自动检测系统主题偏好
  - 记住用户选择（localStorage）
  - 流畅的主题切换动画

#### 顶部导航栏
- Logo 和标题采用纯色设计
- Tab 切换采用 iOS 风格（选中时浅蓝色背景 + 蓝色文字）
- 主题切换按钮带旋转动画
- 返回前台按钮
- 响应式设计（移动端隐藏文字，只显示图标）

#### 图片管理页面
- **操作栏**
  - 圆角卡片设计（16px）
  - 输入框和选择器采用 Apple 风格（10px 圆角）
  - 主按钮蓝色 + 阴影效果
  - hover 时上移 + 阴影增强

- **表格样式**
  - 表头大写字母 + 间距
  - 缩略图圆角（10px）
  - 按钮圆角（8px）
  - Tag 圆角（6px）+ 半透明背景
  - 分页按钮圆角优化
  - 点击缩略图使用 Element Plus 原生图片查看器

- **对话框**
  - 大圆角（20px）
  - 表单输入框 10px 圆角
  - focus 时蓝色边框 + 光晕效果
  - 预览卡片优化（移除 hover 动画，避免闪烁）
  - 简化原图预览（移除 HDR 特殊渲染，统一使用 el-image）

#### 分类管理页面
- **卡片网格**
  - 圆角（16px）
  - hover 时上移 4px + 边框变蓝
  - 图标渐变背景 + 阴影
  - 图标 hover 时放大 + 旋转

- **下拉菜单**
  - 圆角（12px）
  - 菜单项圆角（8px）
  - hover 时背景变化

#### 全局样式优化
- 所有按钮统一圆角（10px）
- 所有卡片统一圆角（16px）
- 所有输入框统一圆角（10px）
- 统一的阴影系统
- 统一的 hover/active 效果
- 完美的深色模式适配
- Element Plus 组件全局样式覆盖（选择器、输入框、对话框、分页、下拉菜单等）

#### 响应式优化
- 移动端单列布局
- 操作栏自适应
- 预览网格自适应
- 按钮全宽显示
- 导航栏自适应（移动端隐藏文字）

### 🐛 Bug 修复
- 修复图片预览卡片 hover 动画导致的闪烁问题
- 修复缩略图 hover 放大导致的预览闪烁问题
- 修复图片查看器被表格覆盖的问题（添加 preview-teleported 属性）
- 修复原图预览显示比例问题（统一使用 fit="contain"）
- 移除自定义预览对话框，改用 Element Plus 原生图片查看器

### 📝 重要说明
- **Live Photo 和 HDR 支持**
  - ✅ 已测试：vivo X200 Pro（Live Photo 和 HDR）
  - ⚠️ 其他安卓手机（小米、OPPO、三星、Google 等）需自行测试
  - ❌ iPhone Live Photo 和 HDR 暂不支持

  - 增强的模糊效果（60px blur + 200% 饱和度）

- **多种滚动方式支持**
  - 鼠标滚轮：垂直滚动转换为横向滚动（0.8 倍速度）
  - 触摸板手势：双指左右滑动

  - 触摸屏：手指左右滑动
  - 完全隐藏滚动条

- **移动端手势支持**
  - 图片左右滑动切换照片
  - 最小滑动距离 50px
  - 判断横向滑动（避免与垂直滚动冲突）

- **流畅的切换动画**
  - 方向性滑动动画（50px 位移）
  - 淡入淡出效果
  - 0.35s 过渡时间
  - cubic-bezier(0.4, 0, 0.2, 1) 缓动
  - 根据切换方向自动选择动画

  - PC 端多张模式：上 80px / 下 150px

  - 移动端多张模式：上 70px / 下 100px

  - 选中边框：白色半透明
  - 悬停效果：图片轻微放大

- **移动端详情面板优化**
  - 从底部弹出，覆盖在缩略图导航栏上方

  - 完全向后兼容（首页不受影响）

  - 防止图片被拖拽（pointer-events: none）
  - 优化触摸事件处理
  - 减少不必要的重渲染

## [1.5.0] - 2024-12-19

### ✨ 新增功能

- **策略模式重构** - 使用设计模式重构照片处理逻辑
  - 创建 `photoProcessors` 模块，每种格式独立处理器
  - `VivoMotionPhotoProcessor` - vivo Live Photo 处理
  - `GoogleMotionPhotoProcessor` - Google/Samsung Motion Photo 处理
  - `IPhoneLivePhotoProcessor` - iPhone Live Photo 处理
  - `HDRPhotoProcessor` - HDR 照片处理
  - `NormalPhotoProcessor` - 普通照片处理（兜底）
  - `PhotoProcessorFactory` - 工厂类自动选择处理器
  - 易于扩展新格式，符合开闭原则

- **HDR 照片渲染** - 使用 gainmap-js 正确渲染 HDR 照片
  - 自动检测 Gain Map 元数据（`Item:Semantic="GainMap"`）
  - WebGL 渲染高亮度图像
  - 支持 vivo X200 Pro Ultra HDR 照片
  - 可配置亮度增强倍数（默认 2.0）
  - 上传时保留 HDR 元数据（Sharp withMetadata）
  - 前端检测结果传递给后端，避免重复检测

- **EXIF 信息恢复** - 完整的 EXIF 提取和显示
  - 自动提取相机型号、焦距、光圈、快门、ISO
  - GPS 坐标自动转换为地名
  - 表单自动填充 EXIF 信息
  - 独立字段存储（model, focal_length, aperture, shutter_speed）

- **局域网访问** - 支持局域网设备访问
  - Vite 监听 `0.0.0.0`
  - 后端监听所有网络接口
  - 启动时自动显示局域网 IP
  - 方便手机测试

### 🎨 界面优化

- **上传对话框优化**
  - 改为按钮式上传（移除大拖拽框）
  - 照片类型标签显示（Live Photo、HDR、来源）
  - EXIF 信息卡片式布局（灰色背景，2行布局）

## [1.0.0] - 2025-12-06

这是 Plog 的第一个正式版本，包含完整的个人摄影作品展示功能。

### ✨ 新增功能

#### 前端展示
- **首页** - 瀑布流照片墙，支持紧凑/广阔视图切换
- **作品集** - 分类筛选，统计卡片，分类描述横幅
- **关于我** - 个人简介，摄影器材，拍摄足迹地图
- **联系方式** - 微信二维码，邮箱联系
- **照片弹窗** - 高清原图查看，EXIF 信息展示，全屏模式

#### 管理后台
- **图片管理** - 上传、编辑、删除照片
- **分类管理** - 添加、编辑、删除分类
- **自动缩略图** - 上传时自动生成 800px 缩略图
- **EXIF 提取** - 自动读取相机信息和 GPS 位置
- **文件同步删除** - 删除记录时同步删除图片文件

#### 配置系统
- **统一配置文件** - `src/config/index.js` 集中管理所有配置
- **配置分类** - 站点配置、API 配置、路径配置、页面配置等
- **构建时注入** - index.html 自动读取配置（vite-plugin-html）
- **SEO 文件生成** - robots.txt 和 sitemap.xml 自动生成

#### SEO 优化
- **动态 Meta** - 每个页面独立的标题、描述、关键词
- **Open Graph** - 微信分享优化
- **结构化数据** - JSON-LD 支持
- **Canonical URL** - 规范链接

#### 性能优化
- **图片懒加载** - 延迟加载提升首屏速度
- **代码分割** - 管理后台独立打包
- **缩略图分离** - 列表用小图，详情用原图

### 🔧 技术实现

- Vue 3 + Composition API
- Vite 5 构建工具
- Tailwind CSS 原子化样式
- Element Plus 管理后台 UI
- Express 后端服务
- Sharp 图片处理
- exifr EXIF 读取
- 高德地图 API GPS 逆地理编码
